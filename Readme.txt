---------------------------------------------------------- Instructions for execution ----------------------------------------------------------

1. Unzip the folder "Q3"
2. Unzipped folder will contain:
    a) Q3.cpp
    b) ReadMe.txt
    c) makefile
3. Open the Q3.cpp in any IDE
4. Open the terminal, run "g++ Q3.cpp -o <output filename>" or "make".

-------------------- Input Format

1. Enter input and output fielname. (Avoid entering file extension. Default file extension is "txt")
2. Output file (i.e., file written after execution of the program) will be in the same folder with the specified name.

-------------------- IMPORTANT

1. All the commands in the assignment pdf is supported with limitiations.
2. No space contained words can be used for file name .i.e, 
    ex: mf file name "Hello worlds"  ====> Not supported
    ex: mf filename "Hello worlds"    ====> Supported
